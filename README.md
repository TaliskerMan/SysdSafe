# SysdSafe

SysdSafe is a systemd service security auditing and hardening tool built with Flutter for Linux systems.

## Documentation

For full installation instructions, setup, and safe usage guidelines, please see the [User Guide](docs/USER_GUIDE.md).

## Support

If you encounter any issues, SysdSafe features a built-in logging facility and an "Email Support" button to quickly get in touch with our team. Please refer to the User Guide for more details.
